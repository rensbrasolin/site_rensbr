Dashboard - Projeto 1


Este dashboard foi criado com o objetivo de explorar recursos de algumas ferramentas de dados, em especial, o Power BI. Embora os dados apresentados sejam fictícios e criados por mim, fiz questão de apresentá-los de maneira coerente para que fosse possível compreender a essência das informações.

O dashboard possui três páginas: Vendas, Clientes/Fornecedores e Funcionários, e cada uma delas apresenta gráficos e informações relevantes sobre uma empresa fictícia que vende games, filmes, séries e livros.

Para uma melhor compreensão dos dados apresentados, sugiro que os títulos dos gráficos sejam lidos com atenção.




Ferramentas utilizadas na criação desse dashboard:

  • Excel: Criação da base dados.
  • Jupyter Notebook: Edição do código.
  • Python: Linguagem de programação.
  • Python (Pandas): Tratamento e análise de dados.
  • Power Point: Criação do layout do dashboard.
  • Power BI: Criação de medidas e colunas, bem como dos gráficos e elementos do dashboard.
