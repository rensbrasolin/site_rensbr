Dashboard de Pagamentos


Nesse dashboard de pagamentos, utilizei o Power BI para tratar e analisar a base de dados fictícia fornecida pela Hashtag Treinamentos. Consultei o Chat GPT quando precisei de ajuda na construção dos códigos de tratamento e modelagem de dados (linguagens 'M' e 'DAX').

A base de dados de pagamentos é coerente, porém não é a mais abrangente. Por não fornecer dados de contas futuras, não é possível fazer análises de contas a pagar, por exemplo. Então, as análises desse dashboard são focadas nas contas pagas.

Esse dashboard tem apenas uma segmentação, que filtra os anos. Porém, consegui distribuir muitas informações dentro de uma mesma página. É importante ler os títulos dos gráficos para melhor interpretação dos dados.




Ferramentas utilizadas na criação desse dashboard:

  • Excel: Visualização prévia da base dados.
  • Power Point: Criação do layout do dashboard.
  • Power BI: Tratamento e análise de dados, e criação dos gráficos e elementos do dashboard.
