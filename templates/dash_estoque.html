Dashboard de Estoque


A base de dados utilizada para esse dashboard foi disponibilizada para treinamento pela Hashtag Treinamentos. Embora eu tenha mais familiaridade com a manipulação de dados no Excel e no Pandas, minha intenção aqui foi utilizar o Power BI tanto para tratar os dados quanto para fazer as análises. Assim, a construção desse dashboard me possibilitou adquirir mais experiência utilizando a linguagem 'M' no Power Query e a linguagem 'DAX' na modelagem de dados.

Em comparação à base de dados utilizada para o meu primeiro dashboard, essa base de dados de estoque ofereceu maiores possibilidades de análise. No entanto, como não havia um objetivo específico na análise desses dados, priorizei análises de movimentação dos produtos em detrimento de análises de valores financeiros.

Esse dashboard possui duas segmentações: uma por ano e outra por subcategoria de produto. Os dados simulam uma empresa varejista do setor de vestuário. Para melhor interpretar os gráficos, sugiro que leiam os títulos de cada um deles.





Recomendo a utilização de tela cheia para obter uma melhor visualização do dashboard.


Ferramentas utilizadas na criação desse dashboard:

  • Excel: Visualização prévia da base dados.
  • Power Point: Criação do layout do dashboard.
  • Power BI: Tratamento e análise de dados, e criação dos gráficos e elementos do dashboard.
