Automação - Conferir Fechamento


Esta é uma automação que utilizo no meu trabalho e obtive prévia autorização para compartilhar. Trata-se de uma tarefa comum que certamente faz parte do dia a dia de muitas empresas. Embora seja simples, dependendo do volume de notas fiscais, pode ser demorada.

A situação é a seguinte: o fornecedor agrupa as notas fiscais faturadas a cada quinzena e emite a cobrança de todas elas em um ou mais boletos. A tarefa consiste em partir da lista de notas fiscais (arquivo no 'Planilhas Google') que compõem o(s) boleto(s), acessar cada uma delas no sistema para obter seus valores, em seguida somá-los e, por fim, dividir o total pelo número de boletos. Se o valor do(s) boleto(s) coincidir com o resultado obtido no código, então a conferência do fechamento está correta.

Para essa automação inicial, optei por utilizar a biblioteca Selenium do Python, que cria um navegador e o controla. Uma das vantagens é que o navegador pode operar em segundo plano, permitindo que o computador seja utilizado para outras tarefas simultaneamente.


Conferir valores das notas fiscais



Outra etapa da tarefa de conferência de fechamento envolve a alteração do status de cada nota fiscal, permitindo-nos ter controle sobre o que já foi faturado e o que ainda não foi. O processo consiste em utilizar a mesma lista de notas fiscais (arquivo no 'Planilhas Google'), acessar o sistema na mesma tela e, para cada nota fiscal, alterar o status de 'pedido' para 'compra'. Ao final, o código gera uma tabela (dataframe) que informa quais notas fiscais tiveram seus status alterados.

Embora fosse possível incorporar esse processo à automação anterior (alterando o status ao obter o valor das notas fiscais), optei por utilizar outra ferramenta de automação, o PyAutoGUI. Diferente do Selenium, o PyAutoGUI é uma espécie de robô (RPA) que permite efetivamente automatizar o uso do mouse e do teclado.

Um último detalhe importante, é que para manter a dinâmica do vídeo e evitar cansaço, fiz uma edição removendo um trecho da gravação.


Alterar status das notas fiscais



Essa automação não apenas libera os profissionais para que se dediquem a tarefas mais subjetivas, mas também proporciona um notável aumento na velocidade e precisão dos processos. Como ilustrado nos exemplos acima, é possível automatizar praticamente qualquer processo repetitivo que faça parte da rotina empresarial, como o envio de relatórios por e-mail para os gestores, a inserção e coleta de dados do sistema de gestão e muitos outros processos.


Ferramentas utilizadas no projeto:

  • Planilhas Google: Arquivo cotendo a relação com os números das notas fiscais que devem ser conferidas.
  • Jupyter Notebook: Edição do código.
  • Python: Linguagem de programação.
  • Python (Pandas): Leitura do arquivo com a relação de notas fiscasis, tratamento e visualização dos dados.
  • Python (Selenium): Controle do Google Chrome na automação web "Conferir Valor".
  • Python (PyAutoGUI): Controle do mouse e do teclado na automação RPA "Alterar Status".
